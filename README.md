# Amazon Homepage Clone (Frontend UI)
A responsive front-end clone of the Amazon website built using HTML and CSS.
This project focuses on recreating the layout and design of Amazon's homepage to practice front-end development skills.

 *Project Overview*
This project is a static clone of the Amazon homepage.
The main goal was to improve my understanding of:

  - Page layout structuring
  - Flexbox positioning
  - Responsive design
  - CSS styling and alignment
  - Real-world UI recreation

 *Features*
 -->Navigation bar with logo, search bar, and menu.
 -->Product category sections
 -->Banner/hero section
 -->Product cards layout
 -->Responsive design structure
 -->Styled using pure CSS
 Note: This is a frontend design project only. No backend functionality or real purchasing system is implemented.

~CSS
